Currency Converter

Автор: [Диляра Нуруллина]

Описание

Приложение для конвертации валют с использованием внешнего API и сохранением истории операций.

Как получить API‑ключ

1. Перейдите на [exchangerate-api.com](https://www.exchangerate-api.com).
2. Зарегистрируйтесь и получите бесплатный API‑ключ.
3. Замените `YOUR_API_KEY` в коде на ваш ключ.
